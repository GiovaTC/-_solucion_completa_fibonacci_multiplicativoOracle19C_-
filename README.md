# Solución completa — Fibonacci Multiplicativo con Oracle 19c .

Documento técnico en formato **Markdown**, listo para uso académico o profesional. La solución está desarrollada en **Java (consola)** y es **100% ejecutable en IntelliJ IDEA**.

---

## Objetivo del programa

* Generar **15 números de 4 dígitos** siguiendo una **serie tipo Fibonacci**.
* Multiplicar los valores generados **en orden secuencial**.
* Mostrar el **resultado final en consola**.
* Registrar el resultado en **Oracle 19c** mediante un **Stored Procedure (SP)**.

---

## 1. Lógica aplicada — Fibonacci multiplicativo

* Se definen **dos valores iniciales** de 4 dígitos.
* Cada nuevo número se genera como la **suma de los dos anteriores**.
* Cada valor se **normaliza** para mantener exactamente **4 dígitos**.
* Todos los valores se **multiplican secuencialmente**.
* El resultado final se **persiste en la base de datos**.

---

## 2. Base de datos Oracle 19c

### 2.1 Tabla de registro

```sql
CREATE TABLE FIBONACCI_PRODUCT_LOG (
    ID NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    TOTAL_NUMBERS NUMBER,
    RESULT_VALUE NUMBER,
    CREATED_AT TIMESTAMP DEFAULT SYSTIMESTAMP
);
```

### 2.2 Stored Procedure

```sql
CREATE OR REPLACE PROCEDURE INSERT_FIBONACCI_PRODUCT (
    p_total_numbers IN NUMBER,
    p_result_value  IN NUMBER
) AS
BEGIN
    INSERT INTO FIBONACCI_PRODUCT_LOG (
        TOTAL_NUMBERS,
        RESULT_VALUE
    ) VALUES (
        p_total_numbers,
        p_result_value
    );

    COMMIT;
END;
/
```

### 3.2 Código Java completo

```java
import java.math.BigInteger;
import java.sql.CallableStatement;
import java.sql.Connection;
import java.sql.DriverManager;

public class FibonacciMultiplicationApp {

    // Ajustar según su entorno
    private static final String DB_URL =
            "jdbc:oracle:thin:@localhost:1521/orcl";
    private static final String DB_USER = "USUARIO_DB";
    private static final String DB_PASSWORD = "PASSWORD_DB";

    public static void main(String[] args) {

        final int TOTAL_NUMBERS = 15;
        int[] fibonacci = new int[TOTAL_NUMBERS];

        // Valores iniciales de 4 dígitos
        fibonacci[0] = 1234;
        fibonacci[1] = 2345;

        // Generación Fibonacci ajustada a 4 dígitos
        for (int i = 2; i < TOTAL_NUMBERS; i++) {
            fibonacci[i] = fibonacci[i - 1] + fibonacci[i - 2];
            fibonacci[i] = normalizeToFourDigits(fibonacci[i]);
        }

        // Multiplicación total (BigInteger para evitar overflow)
        BigInteger product = BigInteger.ONE;

        System.out.println("Serie Fibonacci (4 dígitos):");

        for (int value : fibonacci) {
            System.out.print(value + " ");
            product = product.multiply(BigInteger.valueOf(value));
        }

        System.out.println("\n\nResultado de la multiplicación:");
        System.out.println(product);

        // Registro en Oracle
        saveResultToDatabase(TOTAL_NUMBERS, product);
    }

    private static int normalizeToFourDigits(int value) {
        while (value > 9999) {
            value /= 10;
        }
        if (value < 1000) {
            value += 1000;
        }
        return value;
    }

    private static void saveResultToDatabase(int totalNumbers, BigInteger result) {

        String sql = "{ call INSERT_FIBONACCI_PRODUCT(?, ?) }";

        try (Connection conn = DriverManager.getConnection(
                DB_URL, DB_USER, DB_PASSWORD);
             CallableStatement cs = conn.prepareCall(sql)) {

            cs.setInt(1, totalNumbers);
            cs.setBigDecimal(2, result.toBigDecimal());
            cs.execute();

            System.out.println("\nResultado registrado correctamente en Oracle.");

        } catch (Exception e) {
            System.err.println("Error al registrar en la base de datos:");
            e.printStackTrace();
        }
    }
}
```
